# ProgressHub - Weekly Report Generator

**ProgressHub** is a Next.js-based Weekly Report Generator with AI-powered insights, designed to streamline employee progress tracking and reporting. 

## 🚀 Features
- **Next.js 14 Framework:** Fast, SEO-friendly, and highly optimized web application.
- **NextAuth Integration:** Secure authentication and authorization for users.
- **Interactive Dashboards:** Built with `Chart.js` and `react-chartjs-2` for data visualization.
- **Modern UI Components:** Includes `lucide-react` icons for a clean and responsive interface.

## 🛠️ Tech Stack
- **Frontend/Backend:** [Next.js](https://nextjs.org/) (React 18)
- **Authentication:** [NextAuth.js](https://next-auth.js.org/)
- **Charts:** [Chart.js](https://www.chartjs.org/) & [react-chartjs-2](https://react-chartjs-2.js.org/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Security:** `bcryptjs` for secure password hashing

## 📦 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🐳 Docker Support

The project includes a `Dockerfile` and `docker-compose.yml` for containerized deployment.

To run the application using Docker Compose:

```bash
docker-compose up --build
```
